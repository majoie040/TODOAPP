Voor dit project moest ik een AI-opdracht uitvoeren waarbij ik alleen prompts mocht gebruiken en zelf geen code mocht schrijven. Ik heb gekozen om een to-do app te maken. Het idee was om via verschillende prompts de AI stap voor stap de functionaliteiten te laten opbouwen, zoals taken toevoegen, verwijderen en overzichtelijk weergeven.
Over het algemeen ging het project best goed. Ik merkte dat het werken met alleen prompts best creatief is, omdat je heel duidelijk moet uitleggen wat je wilt. Soms moest ik mijn prompts meerdere keren aanpassen voordat ik het gewenste resultaat kreeg. Dat hielp me wel om beter te leren denken in stappen en duidelijkere instructies te geven.
Wat minder goed ging, is dat mijn account twee keer geblokkeerd werd tijdens het proces. Dat zorgde voor vertraging en wat frustratie, omdat ik daardoor tijdelijk niet verder kon werken. Uiteindelijk heb ik de oude chats kunnen kopieren .
Als ik terugkijk, heb ik vooral geleerd hoe belangrijk precieze prompts zijn en hoe je met AI kunt samenwerken zonder zelf te programmeren. Volgende keer zou ik mijn prompts nog gestructureerder maken en beter letten op de voorwaarden om problemen zoals een blokkade te voorkomen.
